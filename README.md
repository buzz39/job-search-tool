# BooleanJobs.xyz – Job Search Query Builder

🔗 **Live:** [https://booleanjobs.xyz](https://booleanjobs.xyz)

Free tool to create powerful Boolean search queries for finding jobs on LinkedIn, Indeed, Glassdoor, and more. No sign-up required.

## Features

- Build Boolean job search queries instantly
- Target LinkedIn, Indeed, Glassdoor, and other platforms
- SEO-optimized blog with job search tips
- AdSense monetization on blog pages
- Fully responsive design

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Domain:** booleanjobs.xyz

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Deployment

Deployed on Vercel with auto-deploy on push to `main` branch.
